# docker-kubernetes-lab

Dette repositoryet brukes til å øve på Docker og Kubernetes.

## Formål
Repoet er et personlig læringsprosjekt der jeg:
- frisker opp Docker-grunnleggende
- lærer hvordan Kubernetes fungerer i praksis
- deployer containeriserte applikasjoner lokalt
- tester enkel CI med GitHub Actions

## Teknologi
- Docker
- Kubernetes (kind / minikube)
- GitHub Actions
- YAML

## Struktur (under arbeid)

- docker/
- app/
- k8s/
- .github/workflows/

## Status
Aktivt læringsprosjekt. Innhold og struktur vil endre seg etter hvert som jeg lærer mer.
